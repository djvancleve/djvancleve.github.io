HTML Web Structures

Basic Webpage Breakdown

Code Demonstration

                        
<!DOCTYPE html>
<html>
    <head>
        <title>My Webpage</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>

Walkthrough

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  • The <html> element is the root element of an HTML page and has a lang attribute that specifies the language of the document.
  • The <head> element contains meta information about the HTML page, such as the character encoding and the title.
  • The <title> element specifies a title for the HTML page, which is shown in the browser’s title bar or in the page’s tab.
  • The <body> element defines the document’s body and is a container for all the visible contents, such as headings, paragraphs, images, links, etc.
  • The <h1> element defines a large heading.
  • The <p> element defines a paragraph.
